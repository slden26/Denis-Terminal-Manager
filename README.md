# Denis-Terminal-Manager
Портативний менеджер підключень і утиліт. Дозволяє зберігати всі інструменти адміністрування в одній папці і запускати їх через зручний інтерфейс.

##Як підготувати середовище:
###Для коректної роботи програми створіть у кореневій папці наступні директорії та помістіть у них відповідні файли:

- **terminal/ — сюди покладіть wt.exe (Windows Terminal).**
- **ssh/ — сюди покладіть ssh.exe.**
- **calc/ — сюди покладіть cal1.exe (класичний калькулятор).**
- **notepad/ — сюди покладіть портативну версію notepad++.exe.**

Запуск та збірка:
Встановлення залежностей: pip install customtkinter.

Запуск скрипта: python main.py.

Збірка в EXE: Використовуйте PyInstaller з прапорцем --onefile.

Особливості:
Всі шляхи відносні (Portable версія).

Збереження SSH-закладок у локальний config.json.

Швидкий доступ до CMD, PowerShell та системних утиліт.
